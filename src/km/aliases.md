# Aliases

This file contains all aliases. Sending the alias will act identically to the original command.

| Alias          | Original       |
| -------------- | -------------- |
| m              | km.move        |

## Examples

### Using the m alias

Input:
```python
m(5, 10)     # This is equivalent to km.move(5, 10)
```

Output:
```python
m(5, 10)
>>>
```
