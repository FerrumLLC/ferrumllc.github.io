# DHZBox Style API

Ferrum's Software API supports DHZBox Style commands via the UDP Socket. Simply connect via the details listed in the
`Net API` tab in the Ferrum App, and send dhzbox commands as defined in their documentation:
<https://docs.qq.com/doc/DT0xnTVpRenBhT2pQ>. Note that not all commands are supported, but most of the essential ones
are.
